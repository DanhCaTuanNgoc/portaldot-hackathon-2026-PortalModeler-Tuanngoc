# PortalModeler Frontend

Desktop-first UI prototype for the Portaldot local deploy workflow.

## Stack

- Vite
- React
- React Native Web primitives
- Tailwind CSS

## Run

```powershell
cd front-end
npm install
npm run dev
```

Open:

```txt
http://127.0.0.1:5173
```

The current terminal log is local UI state. A later backend bridge can execute the Python scripts in `../scripts` and stream real output into the terminal panel.
